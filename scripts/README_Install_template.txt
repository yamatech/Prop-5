============================================================
Prop-5 v{VERSION} - Installation Guide
============================================================

Thank you for downloading Prop-5!
This package contains the Standalone application and VST3 plugin for Windows (64-bit).

------------------------------------------------------------
1. VST3 Plugin Installation
------------------------------------------------------------
Please copy the 'Prop-5.vst3' folder (the entire folder structure) to the following directory:

  Path: C:\Program Files\Common Files\VST3\

Note:
- If the 'VST3' directory does not exist, please create it.
- After copying, restart your DAW, and it will automatically scan and detect Prop-5.

------------------------------------------------------------
2. Standalone App
------------------------------------------------------------
You can run 'Prop-5.exe' directly from any location. No installation is required.

Prop-5 v{VERSION} - インストール手順
============================================================

Prop-5 をダウンロードいただきありがとうございます！
このパッケージには、Windows用（64bit）のスタンドアロン版アプリとVST3プラグインが含まれています。

------------------------------------------------------------
1. VST3プラグインのインストール
------------------------------------------------------------
以下のディレクトリに 'Prop-5.vst3' フォルダをフォルダごとコピーしてください。

  配置先: C:\Program Files\Common Files\VST3\

注意:
- VST3フォルダが存在しない場合は新規作成してください。
- コピー後、お使いのDAWを再起動すると、自動的にスキャンされて認識されます。

------------------------------------------------------------
2. スタンドアロン版アプリの起動
------------------------------------------------------------
'Prop-5.exe' はインストール不要です。任意の場所に配置してそのまま実行できます。
